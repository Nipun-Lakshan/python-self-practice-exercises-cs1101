# Python Self Practice Exercises - CS1101

A collection of Python self-practice exercises completed as part of the **CS1101 - Fundamentals of Programming** course. This repository is used to improve programming skills through independent practice and problem-solving in Python.

## 📌 About

This repository includes:

* Python self-practice exercises
* Basic programming problems
* Logical thinking tasks
* Small coding challenges
* Fundamental Python implementations

The purpose of this repository is to strengthen my understanding of Python programming by practicing independently and solving a variety of problems.

## 🚀 Topics Covered

* Python Syntax Basics
* Variables and Data Types
* Input and Output
* Operators and Expressions
* Conditional Statements
* Loops
* Functions
* Lists
* Strings
* Dictionaries
* Basic Problem Solving
